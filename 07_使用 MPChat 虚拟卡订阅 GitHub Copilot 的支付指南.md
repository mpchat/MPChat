# 使用 MPChat 虚拟卡订阅 GitHub Copilot 的支付指南

## 准备工作&#x20;

1. **GitHub 账户**：确保账户已验证邮箱并可正常登录

2. **网络环境**：全程使用**美国 IP 地址**（建议住宅 IP）

3. **MPChat 应用**：从官方渠道下载

   * **App Store** | **Google Play** | **Galaxy Store**

   * 官网：**mp.net**

4. **USDT 储备**：建议准备 15−20 等值 USDT（Copilot 个人版 10/月）

## 详细操作步骤&#x20;

### 第 1 步：MPChat 内备款&#x20;

1. **钱包充值**

   * 打开 MPChat →「钱包」→「存款」→ 复制 USDT 地址

   * 从外部钱包/交易所转入 USDT（建议$15+）

2. **申请 MPCard**

   * 进入「MPCard」→「申请新卡」→ **国家必须选择"美国"**

   * 记录卡号、有效期、CVV 码

3. **资产划转**

   * 使用「划转」功能将 USDT 转为美元并充值到 MPCard

   * 确保余额≥$12（包含验证费用）

### 第 2 步：GitHub 账户设置&#x20;

1. 访问[ GitHub 官网](https://github.com/)并登录

2. 点击右上角头像 → **「Settings」**

3. 左侧菜单选择 **「Billing and plans」**

### 第 3 步：订阅 Copilot 服务&#x20;

1. 在 Billing 页面点击 **「Plans」** 选项卡

2. 找到 GitHub Copilot 区域 → 点击 **「Subscribe to Copilot」**

3. 选择个人计划（**Copilot Individual** - $10/月）

4. 点击 **「Continue to payment」**

### 第 4 步：填写支付信息&#x20;

1. **支付方式选择**：选择"Credit card"或"Add new card"

2. **卡片信息填写**：

   * Card number: MPMasterCard 卡号

   * Expiration date: 有效期

   * CVV: 安全码

   * Cardholder name: 建议使用英文名或拼音

3. **账单地址填写（关键步骤）**：

   * Country: **United States**

   * 使用[美国地址生成器](https://www.fakeaddressgenerator.com/)生成完整地址

   * 确保**邮政编码（ZIP）**&#x4E0E;**州（State）**&#x5339;配

   * 示例地址格式：

```plain&#x20;text
123 Main St, Apt 4B
New York, NY 10001
United States
```

1. **完成支付**

   * 勾选同意服务条款

   * 点击 **「Submit」** 提交支付

## 支付验证与激活&#x20;

* GitHub 会进行$1 的小额验证扣款（立即退还）

* 支付成功后，Copilot 服务**立即激活**

* 可在仓库中验证：创建新文件尝试使用 Copilot 建议

## 常见问题解决&#x20;

### ❓ 支付被拒绝&#x20;

**主要原因与解决方案：**

1. **三要素不一致**

   * 确保：美国 IP + 美国 MPCard + 美国账单地址

   * 特别是 ZIP 代码必须真实有效

2. **地址验证失败**

   * 重新生成更详细的美国地址（包含公寓号）

   * 确认州缩写正确（如 CA=加利福尼亚，NY=纽约）

3. **IP 质量问题**

   * 更换为美国住宅 IP（数据中心 IP 易被识别）

### ❓ 扣款成功但服务未激活&#x20;

* 等待 5-10 分钟刷新页面

* 检查邮箱确认 GitHub 发送的订阅确认邮件

* 仍无效时联系[ GitHub 支持](https://support.github.com/)

### ❓ 订阅管理&#x20;

* **查看状态**：Settings → Billing → Plans

* **取消订阅**：同一页面点击"Cancel plan"

* **更换支付方式**：Billing → Payment methods

## 重要提示&#x20;

1. **企业版订阅**：如需 Copilot Business 版（$19/人/月），需由组织所有者统一支付

2. **学生认证**：符合条件用户可[申请免费使用](https://education.github.com/)

3. **MPCard 安全**：建议设置$12 月消费限额，避免意外扣款

4. **发票记录**：所有支付记录可在 Billing 页面下载正式收据

## 费用说明&#x20;

* **个人版**：10/月或 100/年（节省$20）

* **商业版**：$19/人/月

* **7 天免费试用**：新用户可先试用再支付
