# 使用 MPChat 虚拟卡订阅 Cursor Pro 会员支付指南&#x20;

## 订阅前准备&#x20;

1. **MPChat 账户准备**

   * 确保已从官方渠道下载 MPChat 应用

   * 完成身份验证和钱包激活

2. **网络环境配置**

   * 使用稳定的**美国住宅 IP**

   * 避免公共网络或数据中心 IP

3. **资金准备**

   * 准备$25-30 等值 USDT

   * Cursor Pro 月费为$20，需预留验证费用

## 详细操作步骤&#x20;

### 第一阶段：MPChat 虚拟卡准备&#x20;

1. **USDT 充值**

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制您的专属USDT充值地址
- 从外部钱包或交易所转入USDT
- 等待网络确认（通常5-15分钟）
```

1. **申请美国区 MPCard**

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：选择"美国"
- 卡片类型：选择"虚拟卡"
- 立即生成，记录卡号信息
```

1. **资金划转**

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：≥$22（包含验证费用）
```

### 第二阶段：Cursor Pro 订阅流程&#x20;

1. **下载安装 Cursor**

   * 访问[ cursor.sh ](https://cursor.sh/)下载对应版本

   * 或通&#x8FC7;**&#x20;App Store**/**Google Play&#x20;**&#x641C;索安装

2. **启动订阅流程**

   * 打开 Cursor 应用，创建/登录账户

   * 点击左上角设置图标 ⚙️

   * 选择"Upgrade to Pro"

3. **选择订阅方案**

### 第三阶段：支付信息填写（关键步骤）&#x20;

1. **支付界面导航**

   * 选择"Credit Card"支付方式

   * 点击"Add Payment Method"

2. **MPCard 信息填写**

```plain&#x20;text
卡片信息源：MPChat App → MPCard → 卡片详情
- Card Number: 16位卡号
- Expiry Date: 有效期（MM/YY）
- CVC Code: 3位安全码
- Name on Card: 建议使用拼音姓名
```

1. **账单地址生成与填写**

   * 使用专业地址生成器获取真实美国地址

   * 示例格式：

```plain&#x20;text
Street: 1234 Maple Street
City: San Francisco
State: CA
ZIP Code: 94102
Country: United States
```

## 订阅管理指南&#x20;

### 账户管理&#x20;

* **查看状态**：Settings → Billing → Subscription Status

* **下载发票**：Billing 页面可下载正式收据

* **升级/降级**：随时更改订阅计划

### 续费与取消&#x20;

* **自动续费**：默认开启，每月相同日期扣费

* **取消订阅**：Settings → Billing → Cancel Subscription

* **重要提示**：取消后仍可享受已付费周期服务
