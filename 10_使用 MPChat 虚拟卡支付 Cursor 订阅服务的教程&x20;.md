# 使用 MPChat 虚拟卡支付 Cursor 订阅服务的教程&#x20;

## 准备工作&#x20;

1. **Cursor 账户**：下载 Cursor App 并创建账户

2. **网络环境**：全程使用**美国 IP 地址**（建议住宅 IP）

3. **MPChat 应用**：从官方渠道下载

   * **App Store** | **Google Play** | **Galaxy Store**

   * 官网：**mp.net**

4. **USDT 储备**：建议准备 25−30 等值 USDT（Pro 版 20/月）

## 详细操作步骤&#x20;

### 第 1 步：MPChat 内备款&#x20;

1. **钱包充值**

   * 打开 MPChat →「钱包」→「存款」→ 复制 USDT 地址

   * 从外部钱包/交易所转入 USDT（建议$25+）

2. **申请 MPCard**

   * 进入「MPCard」→「申请新卡」→ **国家必须选择"美国"**

   * 记录卡号、有效期、CVV 码

3. **资产划转**

   * 使用「划转」功能将 USDT 转为美元并充值到 MPCard

   * 确保余额≥$22（包含验证费用）

### 第 2 步：Cursor App 内订阅&#x20;

1. **下载安装**

   * 访问[ Cursor 官网](https://cursor.sh/)下载对应版本

   * 或通&#x8FC7;**&#x20;App Store**/**Google Play&#x20;**&#x641C;索"Cursor"下载

2. **启动订阅流程**

   * 打开 Cursor App，登录您的账户

   * 点击左上角**设置图标**（⚙️）

   * 选&#x62E9;**"Upgrade to Pro"**&#x6216;**"Get Cursor Pro"**

### 第 3 步：支付信息填写&#x20;

1. **选择订阅计划**

   * **Monthly**：$20/月

   * **Yearly**：192/年（相当于 16/月，节省 20%）

2. **填写支付信息**

   * 在支付页面选&#x62E9;**"Credit Card"**

   * 填写 MPCard 信息：

     * Card number: MPMasterCard 卡号

     * Expiration date: 有效期（MM/YY）

     * CVC: 安全码

     * Name on card: 建议使用英文名或拼音

3. **账单地址填写（关键步骤）**

   * Country: **United States**

   * 使用[美国地址生成器](https://www.fakeaddressgenerator.com/)生成完整地址

   * 确保地址格式正确：

```plain&#x20;text
456 Oak Avenue, Suite 200
San Francisco, CA 94102
United States
```

1. **完成支付**

   * 确认订单详情

   * 点击 **"Subscribe"** 或 **"Purchase"**

## 支付验证与激活&#x20;

* Cursor 会进行$1 的小额验证扣款（立即退还）

* 支付成功后，Pro 功能**立即激活**

* 可在设置中查看订阅状态为"Active"

## 常见问题解决&#x20;

### ❓ 支付被拒绝&#x20;

**主要原因与解决方案：**

1. **地区不一致**

   * 确保：美国 IP + 美国 MPCard + 美国账单地址

   * 使用 IP 检测网站确认 IP 地理位置

2. **地址验证失败**

   * 重新生成更详细的美国地址

   * 确认城市、州、邮编匹配（如 CA 加州对应邮编范围）

3. **卡片风控**

   * 确保 MPCard 余额充足（≥$22）

   * 尝试更换美国住宅 IP

### ❓ 扣款成功但服务未激活&#x20;

* 重启 Cursor App 并重新登录

* 检查邮箱确认 Cursor 发送的订阅确认邮件

* 联系[ Cursor 支持](https://cursor.sh/contact)

### ❓ 订阅管理&#x20;

* **查看状态**：Cursor 设置 → Billing

* **取消订阅**：同一页面可取消自动续费

* **发票下载**：所有支付记录可下载正式收据
