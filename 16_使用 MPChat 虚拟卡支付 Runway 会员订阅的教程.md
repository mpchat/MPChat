# 使用 MPChat 虚拟卡支付 Runway 会员订阅的教程

## 订阅前准备

1. Runway 账户准备

   * 注册 Runway 官网账户

   * 完成邮箱验证和基本信息设置

2. 网络环境配置

   * 使用美国住宅 IP（关键成功因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备 20−30 等&#x503C;*&#x20;USDT*（*Runway&#x20;*&#x8D77;价 15/月）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$25+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号信息
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：根据选择的Runway套餐确定
```

### 第二阶段：Runway 订阅流程

1. 登录 Runway 账户

   * 访问 runwayml.com

   * 点击右上角"Pricing"或"Upgrade"

2. 选择订阅计划

1) 启动订阅流程

   * 点击选择的计划下的"Get Started"

   * 进入支付信息填写页面

### 第三阶段：支付信息填写

1. 支付界面导航

   * 选择"Credit Card"支付方式

   * 点击"Add Payment Method"

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card Number: 16位虚拟卡号
- Expiration Date: 有效期（MM/YY）
- CVV Code: 3位安全码
- Cardholder Name: 建议使用拼音
```

* 账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国地址生成器

  * 示例格式：

```plain&#x20;text
Address Line 1: 1234 Creative Studio
City: Los Angeles
State: CA
ZIP Code: 90015
Phone: 生成美国电话号码
```

## 故障排查指南

### 支付失败常见原因

1. 地区检测失败

   * 解决方案：使用美国住宅 IP，清除浏览器缓存

2. 地址验证严格

   * 解决方案：使用真实存在的美国地址

3. 风控限制

   * 解决方案：确保 MPCard 余额充足

   * 更换更干净的美国 IP

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 等待 10-15 分钟系统同步

   * 检查注册邮箱确认邮件

   * 联系 Runway 支持

2. 功能访问受限

   * 确认登录的是订阅账户

   * 检查浏览器 Cookie 和缓存

   * 尝试不同浏览器访问
