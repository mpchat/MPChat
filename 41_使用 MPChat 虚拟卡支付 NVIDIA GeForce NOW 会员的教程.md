## 详细操作步骤

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$25+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$15-20（包含验证费用）
```

### 第二阶段：GeForce NOW 订阅流程

1. 访问 NVIDIA 官网

   * 使用美国 IP 访问 geforcenow.com

   * 点击"Join Today"或"Get Started"

2. 选择订阅计划

1) 注册/登录账户

   * 新用户：创建 NVIDIA 账户

   * 现有用户：直接登录

   * 完成邮箱验证

### 第三阶段：支付信息填写

1. 支付界面导航

   * 选择订阅周期（月付或季度付）

   * 点击"Continue to Payment"

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用拼音
```

* 账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国地址生成器

  * 示例格式：

```plain&#x20;text
Address Line 1: 1234 Gaming Street
City: Santa Clara
State: CA
ZIP Code: 95050
Phone number: 生成美国电话号码
```

## 故障排查指南

### 支付失败常见原因

1. 地区不一致

   * 解决方案：确保美国 IP+美国卡+美国地址三位一体

2. 地址验证失败

   * 解决方案：使用更详细的真实美国地址

3. NVIDIA 账户区域锁定

   * 解决方案：创建全新的 NVIDIA 账户

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 等待 10-15 分钟系统同步

   * 联系 NVIDIA 客服

2. 支付被拒绝

   * 更换更干净的美国住宅 IP

   * 重新生成美国账单地址

   * 确保 MPCard 余额充足
