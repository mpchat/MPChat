# 使用 MPChat 虚拟卡开通 Substack 付费订阅的教程

## 流程总览



## 订阅前准备

1. Substack 账户准备

   * 注册或登录 Substack 官网

   * 确保账户已完成基本验证

2. 网络环境配置

   * 使用美国住宅 IP（重要因素）

   * 确保 IP 地理位置显示为美国

3. MPChat 资金准备

   * 准备$10-20 等值 USDT（Substack 订阅价格因作者而异）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$15+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$10-15（根据订阅价格调整）
```

### 第二阶段：Substack 订阅流程

1. 访问目标 Substack 专栏

   * 使用美国 IP 访问您要订阅的 Substack 专栏

   * 示例：某个作家的专栏页面

2. 选择订阅计划

1) 启动订阅流程

   * 点击"Subscribe"或"Upgrade"按钮

   * 选择订阅类型和周期

   * 进入支付信息页面

### 第三阶段：支付信息填写

1. 支付界面导航

   * Substack 使用 Stripe 支付处理器

   * 选择"Credit or debit card"

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVC: 3位安全码
- Name on card: 建议使用拼音
```

1. 账单地址填写（关键步骤）

   * Country: United States

   * 使用美国地址生成器

   * 示例格式：

```plain&#x20;text
Address line 1: 1234 Reading Street
City: San Francisco
State: CA
ZIP code: 94102
```

## 支付验证与激活

1. 即时验证

   * Stripe 会进行小额验证扣款（立即退还）

   * 验证卡片有效性

2. 服务激活

   * 支付成功后，付费内容立即解锁

   * 可以访问订阅者的专属内容

3. 功能验证

   * 检查付费文章访问权限

   * 验证评论区权限

   * 确认邮件通知功能

## 故障排查指南

### 支付失败常见原因

1. 地址验证问题

   * 解决方案：使用更详细的真实美国地址

2. 卡片风控

   * 解决方案：确保 MPCard 余额充足

3. IP 问题

   * 解决方案：使用稳定的美国住宅 IP

### 订阅状态异常处理

1. 扣款成功但内容未解锁

   * 等待几分钟刷新页面

   * 检查确认邮件

   * 联系专栏作者

2. 支付被拒绝

   * 检查卡片信息是否正确

   * 验证账单地址准确性

   * 确认 MPCard 内有足够余额

## 订阅管理指南

### 账户管理

* 查看订阅：Settings → Subscriptions

* 管理支付方式：更新或更换支付卡

* 下载收据：获取支付凭证

### 取消订阅

* 路径：Settings → Subscriptions → Cancel

* 重要提示：取消后仍可访问至付费周期结束

## 费用优化建议

2. 多专栏订阅：合理规划订阅预算

3. 免费试读：先阅读免费内容再决定订阅
