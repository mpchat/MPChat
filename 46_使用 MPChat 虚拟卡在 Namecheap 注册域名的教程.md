# 使用 MPChat 虚拟卡在 Namecheap 注册域名的教程

## 注册前准备

1. 域名信息准备

   * 确定要注册的域名

   * 准备有效的邮箱地址

   * 了解需要的域名后缀（。com, .net, .org 等）

2. 网络环境

   * 稳定网络连接

   * 建议使用美国 IP（提高成功率）

3. MPChat 资金准备

   * 准备$15-25 等值 USDT

   * 域名注册费用通常为$10-15/年

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 转入足额USDT（建议$20+）
- 等待网络确认
```

* 申请国际 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 选择支持的国家（推荐美国）
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$15-20（包含税费）
```

### 第二阶段：Namecheap 域名搜索

1. 访问 Namecheap 官网

   * 打开 namecheap.com

   * 点击顶部搜索框

2. 搜索域名

   * 输入想要注册的域名

   * 查看可用性和价格

   * 选择适合的域名后缀

3. 添加到购物车

   * 点击"Add to Cart"

   * 选择注册年限（1 年起）

   * 确认价格和优惠

### 第三阶段：域名配置

1. 选择基础服务

   * WHOIS 隐私保护（推荐选择）

   * 自动续费设置（根据需求选择）

   * 域名邮箱服务（可选）

2. 配置域名设置

   * 姓名服务器设置（默认或自定义）

   * 联系人信息填写

   * 确认服务条款

## 支付信息填写

1. 支付界面导航

   * 进入购物车结算页面

   * 选择"Credit Card"支付方式

   * 点击"Continue"

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用拼音
```

1. 账单地址填写

   * Country: 选择卡片发行国

   * 地址信息：使用地址生成器

   * 联系电话：可填生成号码

   * 注意：确保信息格式正确

## 支付验证与域名激活

1. 即时支付
