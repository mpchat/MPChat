# 使用 MPChat 虚拟卡开通 DigitalOcean droplets 的教程

## 开通前准备&#x20;

1. **账户准备**

   * 准备有效的电子邮箱

   * 准备手机号码（用于验证）

   * 确定服务器用途和配置需求

2. **网络环境配置**

   * 使用**美国住宅 IP**（重要因素）

   * 确保 IP 干净且地理位置显示为美国

3. **MPChat 资金准备**

   * 准备 20−50 等值 USDT（Droplet 起价 4/月）

   * 建议多预留资金用于验证和预授权

## 详细操作步骤&#x20;

### 第一阶段：MPChat 虚拟卡准备&#x20;

1. **USDT 充值**

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 转入足额USDT（建议$50+）
- 等待网络确认到账
```

* **申请美国区 MPCard**

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* **资金划转**

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$30-40（包含预授权费用）
```

### 第二阶段：DigitalOcean 账户注册&#x20;

1. **访问官网注册**

   * 使用美国 IP 访问 digitalocean.com

   * 点击"Sign Up"开始注册

   * 输入邮箱地址和密码

2. **账户验证**

   * 完成邮箱验证

   * 进行手机号码验证

   * 填写基本个人信息

3. **账户审核**

   * 等待账户审核通过（通常几分钟到几小时）

   * 审核期间可以浏览产品但无法创建资源

### 第三阶段：添加支付方式&#x20;

1. **进入支付设置**

   * 登录后点击右上角头像 → "Billing"

   * 选择"Payment Methods"

   * 点击"Add Credit Card"

2. **MPCard 信息填写**

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用拼音
```

* **账单地址填写（关键步骤）**

  * Country: **United States**

  * 使用专业美国地址生成器

  * 示例格式：

```plain&#x20;text
Address Line 1: 1234 Cloud Street
City: New York
State: NY
ZIP Code: 10001
```

## 支付验证与账户激活&#x20;

1. **预授权验证**

   * DigitalOcean 会进行$5 预授权扣款

   * 验证成功后金额会退还

   * 验证过程需要 1-3 个工作日

2. **账户激活**

   * 支付方式验证通过后账户完全激活

   * 可以开始创建 Droplet 和其他服务

3. **功能验证**

   * 测试创建小型 Droplet

   * 验证 SSH 连接功能

   * 检查控制面板访问

## 创建 Droplet 详细步骤&#x20;

1. **选择创建方式**

   * 登录控制面板 → 点击"Create" → "Droplets"

   * 选择"Quick Create"或"Advanced Options"

2. **配置 Droplet 参数**

   * 选择镜像（操作系统）

   * 选择套餐规格

   * 选择数据中心区域

   * 添加 SSH 密钥（推荐）

3. **最终确认**

   * 设置 Droplet 名称

   * 选择项目归属

   * 确认月度费用

   * 点击"Create Droplet"

## 故障排查指南&#x20;

### 支付验证失败&#x20;

1. **常见原因**

   * 地址信息不匹配

   * 预授权金额不足

   * 账户信息可疑

2. **解决方案**

   * 使用更详细的真实地址

   * 确保 MPCard 余额充足

   * 联系客服说明情况

### 账户审核问题&#x20;

1. **审核不通过**

   * 提供更多验证信息

   * 使用真实个人信息

   * 联系支持团队

2. **验证延迟**

   * 耐心等待 1-3 个工作日

   * 检查邮箱垃圾邮件文件夹

   * 通过在线聊天咨询进度

## 安全最佳实践&#x20;

1. **SSH 密钥认证**

   * 使用 SSH 密钥代替密码登录

   * 禁用 root 密码登录

   * 定期更换密钥对

2. **防火墙配置**

   * 配置 UF 防火墙

   * 只开放必要端口

   * 限制 IP 访问范围

3. **系统更新**

   * 定期更新系统补丁

   * 安装安全更新

   * 监控安全日志
