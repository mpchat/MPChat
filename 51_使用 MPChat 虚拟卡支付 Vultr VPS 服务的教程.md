# 使用 MPChat 虚拟卡支付 Vultr VPS 服务的教程

## 开通前准备

1. 账户准备

   * 准备有效的电子邮箱

   * 准备手机号码（用于验证）

   * 确定 VPS 配置需求

2. 网络环境配置

   * 使用美国住宅 IP（重要因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备$10-30 等值 USDT（Vultr 按小时计费，需先充值）

   * 建议充值$20-50 作为启动资金

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 转入足额USDT（建议$50+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$25-40（用于账户充值）
```

### 第二阶段：Vultr 账户注册

1. 访问官网注册

   * 使用美国 IP 访问 vultr.com

   * 点击"Create Account"或"Sign Up"

   * 输入邮箱地址和密码

2. 账户验证

   * 完成邮箱验证

   * 进行手机号码验证（可选）

   * 填写基本个人信息

3. 账户审核

   * Vultr 通常即时审核通过

   * 审核通过后即可进行充值

### 第三阶段：账户充值和支付验证

1. 进入充值页面

   * 登录后点击右上角"Billing"

   * 选择"Make Payment"

   * 点击"Add Funds"

2. 选择充值金额

3. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用拼音
```

* 账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国地址生成器

  * 示例格式：

```plain&#x20;text
Address Line 1: 1234 Cloud Avenue
City: Seattle
State: WA
ZIP Code: 98101
```

## 部署 VPS 详细步骤

1. 选择部署方式

   * 登录控制面板 → 点击"+"按钮

   * 选择"Deploy New Server"

2. 选择服务器类型

   * Cloud Compute（标准云服务器）

   * Bare Metal（裸金属服务器）

   * Block Storage（块存储）

3. 选择服务器位置

   * 根据用户群体选择最近的位置

   * 测试不同位置的网络延迟

   * 考虑数据合规要求

4. 选择操作系统

   * Linux 发行版（免费）

   * Windows Server（额外收费）

   * 自定义 ISO（高级用户）

5. 选择服务器规格

   * CPU 核心数

   * 内存大小

   * 存储类型和容量

   * 流量配额

6. 其他配置

   * 服务器主机名

   * SSH 密钥添加

   * 启动脚本设置

   * 防火墙规则

7. 最终部署

   * 确认月度费用

   * 点击"Deploy Now"

   * 等待服务器初始化（5-10 分钟）

## 故障排查指南

### 支付验证失败

1. 常见原因

   * 地址信息不匹配

   * 卡片风控限制

   * IP 地址可疑

2. 解决方案

   * 使用更详细的真实地址

   * 确保 MPCard 余额充足

   * 更换干净的美国住宅 IP

### 账户审核问题

1. 审核不通过

   * 提供真实个人信息

   * 使用美国地址和电话

   * 联系客服说明情况

2. 验证延迟

   * 通常即时通过，如延迟可联系客服

   * 检查邮箱垃圾邮件文件夹

   * 通过在线聊天咨询进度
