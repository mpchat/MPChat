# 使用 MPChat 虚拟卡开通 Google Cloud Platform （GCP） 的教程

## 开通前准备

1. Google 账户准备

   * 拥有一个 Google 账户（Gmail 账户）

   * 账户最好没有开通过 GCP 免费试用

2. 网络环境配置

   * 使用美国住宅 IP（关键成功因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备$10-20 等值 USDT（GCP 会进行 1 美元预授权验证，免费试用通常不需要支付，但需要验证卡片）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

* 操作路径：MPChat App → 钱包 → 存款
  复制 USDT 充值地址
  从外部钱包转入足额 USDT（建议$20+）
  等待网络确认到账

1. 申请美国区 MPCard

* 操作路径：MPChat App → MPCard → 申请新卡
  发行国家：必须选择"美国"
  卡片类型：虚拟卡（即时生成）
  安全记录卡号、有效期、CVV 码

1. 资金划转

* 操作路径：MPChat App → 钱包 → 划转
  从：MP 钱包（USDT）
  到：MPCard（美元）
  金额：$10-15（包含验证费用）

### 第二阶段：GCP 账户注册

1. 访问 GCP 控制台

   * 使用美国 IP 访问 cloud.google.com

   * 点击"Get started for free"

2. 登录 Google 账户

   * 使用您的 Google 账户登录

   * 如果已有 GCP 项目，可能需要新建一个项目

3. 开始免费试用

   * 点击"Start free trial"

   * 阅读并同意服务条款

### 第三阶段：支付信息填写

1. 支付界面导航

   * 选择"个人"或"企业"（根据实际情况选择，个人用户选择个人）

   * 填写个人信息（姓名、地址等）

2. MPCard 信息填写

* 信息源：MPChat App → MPCard → 卡片详情
  Card number: 16 位虚拟卡号
  Expiration date: 有效期（MM/YY）
  CVV: 3 位安全码
  Name on card: 建议使用拼音

1. 账单地址填写（关键步骤）

   * Country: United States

   * 使用专业美国地址生成器

   * 示例格式：

2. Address line 1: 1234 Cloud Street
   City: Mountain View
   State: CA
   ZIP code: 94043

   * 重点验证：加州邮编范围 90001-96162

## 支付验证与激活

1. 即时验证

   * GCP 会进行 1 美元预授权扣款（立即退还）

   * 验证卡片有效性和地址匹配度

2. 服务激活

   * 支付验证成功后，GCP 免费试用立即激活

   * 获得$300 赠金，有效期 90 天

3. 功能验证

   * 登录 GCP 控制台

   * 创建第一个 VM 实例或存储桶

   * 验证服务是否正常

## 故障排查指南

### 支付失败常见原因

1. 地区不一致

   * 解决方案：确保美国 IP+美国卡+美国地址三位一体

2. 地址验证失败

   * 解决方案：使用更详细的真实美国地址

3. Google 账户区域问题

   * 解决方案：Google 账户的地区设置为美国

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 等待 10-15 分钟系统同步

   * 检查注册邮箱确认邮件

   * 联系 Google Cloud 支持

2. 支付被拒绝

   * 更换更干净的美国住宅 IP

   * 重新生成美国账单地址

   * 确保 MPCard 余额充足
