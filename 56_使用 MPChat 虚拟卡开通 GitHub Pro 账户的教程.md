# 使用 MPChat 虚拟卡开通 GitHub Pro 账户的教程

## 订阅前准备

1. GitHub 账户准备

   * 已有 GitHub 账户并完成邮箱验证

   * 确保账户无欠费或异常状态

2. 网络环境配置

   * 使用美国住宅 IP（重要成功因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备 10−15 等&#x503C;*&#x20;USDT*（*GitHubPro&#x20;*&#x6708;费 7，年费$84）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

* 操作路径：MPChat App → 钱包 → 存款
  复制 USDT 充值地址
  从外部钱包转入足额 USDT（建议$15+）
  等待网络确认到账

1. 申请美国区 MPCard

* 操作路径：MPChat App → MPCard → 申请新卡
  发行国家：必须选择"美国"
  卡片类型：虚拟卡（即时生成）
  安全记录卡号、有效期、CVV 码

1. 资金划转

* 操作路径：MPChat App → 钱包 → 划转
  从：MP 钱包（USDT）
  到：MPCard（美元）
  金额：$10-12（包含验证费用）

### 第二阶段：GitHub Pro 订阅流程

1. 登录 GitHub 账户

   * 使用美国 IP 访问 github.com

   * 点击右上角头像 → "Settings"

2. 选择订阅计划

   * 在左侧边栏点击"Billing and plans"

   * 点击"Upgrade"或"Manage plan"

3. 选择 Pro 计划

1) 进入支付页面

   * 选择按月或按年支付（推荐按年节省费用）

   * 点击"Continue to payment"

### 第三阶段：支付信息填写

1. 支付界面导航

   * 选择"Credit card"支付方式

   * 点击"Add credit card"

2. MPCard 信息填写

* 信息源：MPChat App → MPCard → 卡片详情
  Card number: 16 位虚拟卡号
  Expiration date: 有效期（MM/YY）
  CVV: 3 位安全码
  Name on card: 建议使用拼音

1. 账单地址填写（关键步骤）

   * Country: United States

   * 使用专业美国地址生成器

   * 示例格式：

2. Address line 1: 1234 Code Street
   City: San Francisco
   State: CA
   ZIP code: 94107

   * 重点验证：加州邮编范围 90001-96162

## 支付验证与激活

1. 即时验证

   * GitHub 会进行$1 验证扣款（立即退还）

   * 验证卡片有效性和地址匹配度

2. 服务激活

   * 支付成功后，GitHub Pro 立即激活

   * 账户显示 Pro 标识，享受高级功能

3. 功能验证

   * 创建私有仓库（应无数量限制）

   * 验证高级代码分析工具

   * 检查 Actions 使用时间

## 故障排查指南

### 支付失败常见原因

1. 地区检测失败

   * 解决方案：使用美国住宅 IP，清除浏览器缓存

2. 地址验证严格

   * 解决方案：使用真实存在的美国地址

3. 账户风控

   * 解决方案：确保账户有正常使用记录

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 等待 5-10 分钟系统同步

   * 检查注册邮箱确认邮件

   * 联系 GitHub 支持

2. 支付被拒绝

   * 更换更干净的美国住宅 IP

   * 重新生成美国账单地址

   * 确保 MPCard 余额充足

## 订阅管理指南

### 账户管理

* 查看状态：Settings → Billing and plans

* 发票下载：Billing 页面可下载详细发票

* 套餐变更：随时升级或降级套餐

### 续费与取消

* 自动续费：默认开启，周期结束时自动扣费

* 取消订阅：Billing → Cancel plan

* 重要提示：取消后仍可享受至周期结束
