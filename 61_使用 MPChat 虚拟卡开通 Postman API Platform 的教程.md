# 使用 MPChat 虚拟卡开通 Postman API Platform 的教程

## 订阅前准备

1. Postman 账户准备

   * 已有 Postman 账户并完成基础使用

   * 账户邮箱验证完成

2. 网络环境配置

   * 使用美国住宅 IP（重要成功因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备 20−40 等&#x503C;*&#x20;USDT*（基础版 12/用户/月，专业版$29/用户/月）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$50+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$30-50（根据团队规模调整）
```

### 第二阶段：Postman 订阅流程

1. 登录 Postman 账户

   * 使用美国 IP 访问 postman.com

   * 点击右上角"Pricing"或"Upgrade"

2. 选择订阅计划

1) 配置团队设置

   * 选择用户数量（最少 3 用户起）

   * 选择订阅周期（年付有折扣）

   * 配置团队名称和工作区

### 第三阶段：支付信息填写

1. 支付界面导航

   * 选择"Credit Card"支付方式

   * 点击"Add Payment Method"

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用拼音
```

* 账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国地址生成器

  * 示例格式：

```plain&#x20;text
Address Line 1: 1234 API Street
City: San Francisco
State: CA
ZIP Code: 94102
Phone: 生成美国电话号码
```

## 故障排查指南

### 支付失败常见原因

1. 地区检测失败

   * 解决方案：使用美国住宅 IP，清除浏览器缓存

2. 地址验证严格

   * 解决方案：使用真实存在的美国地址

3. 团队规模限制

   * 解决方案：Team 版最少需要 3 个用户

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 等待 5-10 分钟系统同步

   * 检查注册邮箱确认邮件

   * 联系 Postman 支持

2. 功能访问受限

   * 确认登录的是订阅团队账户

   * 清除浏览器缓存重新登录

   * 检查团队成员权限设置

## 订阅管理指南

### 团队管理

* 查看状态：Settings → Billing

* 成员管理：添加/移除团队成员

* 工作区管理：配置团队工作区

### 续费与取消

* 自动续费：默认开启，每年自动续费

* 取消订阅：Billing → Cancel Plan

* 重要提示：取消后仍可享受至周期结束

## 费用优化建议
