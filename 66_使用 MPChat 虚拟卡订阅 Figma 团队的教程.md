# 使用 MPChat 虚拟卡订阅 Figma 团队的教程

## 订阅前准备

1. 网络环境配置

   * 使用美国住宅 IP（关键成功因素）

   * 确保 IP 干净且地理位置显示为美国

2. MPChat 资金准备

   * 准备 100−200 等&#x503C;*&#x20;USDT*（团队版 12/编辑者/月）

   * 建议多预留资金用于验证费用

3. 团队规划

   * 确定团队成员数量

   * 规划项目结构

   * 确定权限需求

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$150+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$100-120（根据团队规模调整）
```

### 第二阶段：Figma 团队设置

1. 访问 Figma 官网

   * 使用美国 IP 访问 figma.com

   * 点击"Sign Up"注册新账户

2. 创建团队

   * 登录后点击左上角"+"图标

   * 选择"Team" → "Create new team"

   * 输入团队名称和描述

3. 选择订阅计划

### 第三阶段：团队配置

1. 设置团队信息

   * 团队名称和 logo

   * 默认项目设置

   * 邀请链接设置

2. 成员管理

   * 添加团队成员邮箱

   * 设置成员角色（编辑者/查看者）

   * 配置权限级别

## 支付信息填写

1. 进入支付页面

   * 团队设置 → Billing

   * 点击"Upgrade to Professional"

   * 选择年付或月付（推荐年付节省 18%）

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用团队名称拼音
```

* 账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国地址生成器

  * 示例格式：

```plain&#x20;text
Company: [您的团队英文名称]
Address: 1234 Design Street
City: San Francisco
State: CA
ZIP Code: 94102
VAT Number: 如有可填写（非必需）
```

## 支付验证与激活

1. 即时验证

   * Figma 会进行$1 验证扣款（立即退还）

   * 验证卡片有效性和地址匹配度

2. 服务激活

   * 支付成功后，团队版功能立即激活

   * 所有编辑者获得完整权限

   * 可以创建无限项目

3. 功能验证

   * 测试团队库功能

   * 验证版本历史记录

   * 检查高级权限设置

## 故障排查指南

### 支付失败常见原因

1. 地区检测失败

   * 解决方案：使用美国住宅 IP，清除浏览器缓存

2. 地址验证严格

   * 解决方案：使用真实存在的美国地址

3. 团队规模限制

   * 解决方案：确保编辑者数量合理

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 等待 5-10 分钟系统同步

   * 检查注册邮箱确认邮件

   * 联系 Figma 支持

2. 功能访问受限

   * 确认登录的是团队管理员账户

   * 清除浏览器缓存重新登录

   * 检查团队成员权限设置
