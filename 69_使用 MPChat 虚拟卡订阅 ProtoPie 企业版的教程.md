# 使用 MPChat 虚拟卡订阅 ProtoPie 企业版的教程

## 订阅前准备

1. 企业信息准备

   * 公司英文名称和地址

   * 企业邮箱（非免费邮箱）

   * 公司网站（如有）

   * 预计用户数量和使用需求

2. 网络环境配置

   * 使用美国住宅 IP（重要成功因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备 2000−5000 等&#x503C;*&#x20;USDT*（企业版起价约 2000/年）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$3000+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
- 建议申请较高额度的卡片
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：根据报价确定（建议$2500+）
```

### 第二阶段：ProtoPie 企业版咨询流程

1. 访问 ProtoPie 官网

   * 使用美国 IP 访问 protopie.io

   * 点击"Pricing"或"Enterprise"

2. 联系销售团队

   * 填写企业信息表格：

     * 公司名称（英文）

     * 企业邮箱

     * 公司规模

     * 预计用户数量

     * 使用场景描述

3. 获取定制报价

   * 等待销售代表联系（通常 1-2 个工作日）

   * 讨论具体需求和定价

   * 接收正式报价单

### 第三阶段：支付流程

1. 接收支付链接

   * 销售代表发送专属支付链接

   * 链接包含定制化报价详情

   * 支付链接有效期通常为 30 天

2. 支付界面导航

   * 通过销售提供的链接进入支付页面

   * 确认订单详情（用户数、功能、期限）

   * 选择"Credit Card"支付方式

## 支付信息填写

1. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用公司名称拼音
```

* 企业账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国企业地址生成器

  * 示例格式：

```plain&#x20;text
Company Name: [您的公司英文名称]
Address Line 1: 1234 Prototype Avenue
City: San Francisco
State: CA
ZIP Code: 94102
Phone: 生成美国企业电话号码
Tax ID: 如有美国税号可提供（非必需）
```

## 支付验证与激活

1. 大额支付验证

   * ProtoPie 可能进行验证电话或邮件确认

   * 准备回答关于企业用途的问题

   * 可能需要提供企业证明文件

2. 服务激活

   * 支付成功后，销售代表会协助激活账户

   * 获得企业管理员权限

   * 收到许可证文件和使用指南

3. 功能验证

   * 测试企业版专属功能

   * 验证团队管理权限

   * 检查高级功能访问

## 故障排查指南

### 支付失败常见原因

1. 企业验证失败

   * 解决方案：提供真实企业信息，准备验证文件

2. 地址验证严格

   * 解决方案：使用真实存在的美国企业地址

3. 风控限制

   * 解决方案：与销售代表沟通支付方式

   * 确保 MPCard 余额充足

### 订阅状态异常处理

1. 扣款成功但服务未激活

   * 立即联系销售代表提供支付凭证

   * 检查邮箱确认邮件

   * 等待人工审核完成（1-3 个工作日）

2. 功能访问问题

   * 确认登录的是企业管理员账户

   * 检查许可证文件状态

   * 联系专属客户经理

## 订阅管理指南

### 企业账户管理

* 许可证管理：企业管理员控制台

* 用户管理：添加/移除团队成员

* 权限设置：配置不同权限级别

### 续费与升级

* 年度合约：通常按年订阅，到期前续费

* 功能升级：联系销售代表添加新功能

* 用户调整：根据需求调整用户数量

## 费用优化建议

1. 年度订阅：选择年付通常有折扣

2. 批量许可：用户数越多单价可能越低

3. 长期合作：签订多年合约获得优惠

## 安全提醒

1. MPCard 保护：设置合理的消费限额

2. 企业信息安全：保护管理员账户

3. 许可证安全：妥善保管许可证文件
