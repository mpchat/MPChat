# 使用 MPChat 虚拟卡订阅 Unity Plus/Pro 的教程

## 订阅前准备

1. 网络环境配置

   * 使用美国住宅 IP（关键成功因素）

   * 确保 IP 干净且地理位置显示为美国

2. MPChat 资金准备

   * 准备 200−2000 等&#x503C;*&#x20;USDT*（*Plus&#x20;*&#x7248; 399/年，Pro 版$1800/年）

   * 建议多预留资金用于验证费用

3. 身份验证准备

   * 准备个人或企业信息

   * 确定订阅用途（个人/企业）

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$2000+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：根据订阅版本确定（$400-2000）
```

### 第二阶段：Unity 账户注册与验证

1. 访问 Unity 官网

   * 使用美国 IP 访问 unity.com

   * 点击"Get started"或"Sign up"

2. 创建 Unity ID

   * 使用企业邮箱注册（推荐）

   * 完成邮箱验证

   * 设置账户密码和安全问题

3. 身份验证

   * 填写个人或企业信息

   * 选择账户类型（个人/企业）

   * 完成基本资料填写

### 第三阶段：选择订阅计划

1. 访问订阅页面

   * 登录后访问 store.unity.com

   * 点击"Plans and pricing"或"Subscribe"

2. 版本对比选择

1) 启动订阅流程

   * 选择"Plus"或"Pro"版本

   * 点击"Subscribe Now"

   * 选择订阅周期（年付/月付）

## 支付信息填写

1. 支付界面导航

   * 选择"Credit Card"支付方式

   * 点击"Add Payment Method"

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用企业名称拼音
```

* 账单地址填写（关键步骤）

  * Country: United States

  * 使用专业美国企业地址生成器

  * 示例格式：

```plain&#x20;text
Company Name: [您的公司英文名称]
Address Line 1: 1234 Game Dev Avenue
City: San Francisco
State: CA
ZIP Code: 94102
Phone: 生成美国企业电话号码
```

## 支付验证与激活

1. 即时验证

   * Unity 会进行小额验证扣款（立即退还）

   * 验证卡片有效性和地址匹配度

2. 许可证激活

   * 支付成功后，许可证立即激活

   * 在 Unity Hub 中查看许可证状态

   * 收到激活确认邮件

3. 功能验证

   * 登录 Unity Hub 验证许可证

   * 测试 Pro 版专属功能

   * 检查云构建服务访问

## 故障排查指南

### 支付失败常见原因

1. 企业验证失败

   * 解决方案：提供真实企业信息，确保一致性

2. 地址验证严格

   * 解决方案：使用真实存在的美国企业地址

3. 收入验证

   * 解决方案：如实填写年收入信息

### 订阅状态异常处理

1. 扣款成功但许可证未激活

   * 等待 10-15 分钟系统同步

   * 检查注册邮箱确认邮件

   * 联系 Unity 支持

2. 许可证验证失败

   * 确认登录的是订阅账户

   * 清除 Unity Hub 缓存重新登录

   * 检查许可证文件状态
