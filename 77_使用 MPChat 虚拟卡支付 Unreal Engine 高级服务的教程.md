# 使用 MPChat 虚拟卡支付 Unreal Engine 高级服务的教程

## 服务概述

Unreal Engine 提供多种付费服务选项，主要包括：

* Unreal Engine 企业版：定制化许可协议

* 定制化服务：技术支持、培训、定制开发

* Marketplace：购买资产、插件、工具

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 转入足额USDT（企业服务通常$1500+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：根据预估费用确定
```

### 第二阶段：Epic Games 账户准备

1. 注册开发者账户

   * 访问 Epic Games 官网

   * 点击"Get Started"创建账户

   * 完成邮箱验证和基本资料

2. 企业身份验证

   * 提供企业信息（公司名称、规模、业务）

   * 验证开发者身份

   * 等待审核通过

### 第三阶段：企业服务申请流程

1. 联系 Epic 销售团队

   * 访问企业服务页面

   * 填写企业需求表格

   * 等待销售代表联系（1-3 个工作日）

2. 需求沟通

   * 说明具体业务需求

   * 讨论定制化服务内容

   * 确定服务级别和价格

### 第四阶段：支付流程

1. 接收定制报价

   * Epic 销售提供详细报价单

   * 包含服务内容和价格明细

   * 确认服务条款

2. MPCard 支付信息

```plain&#x20;text
支付信息填写：
- 卡号：16位MPCard虚拟卡号
- 有效期：MM/YY格式
- CVV：3位安全码
- 持卡人姓名：企业名称拼音
```

* 企业账单地址

```plain&#x20;text
地址格式示例：
公司名称：Your Company Inc.
地址：1234 Tech Avenue
城市：San Francisco
州：CA
邮编：94102
国家：United States
```
