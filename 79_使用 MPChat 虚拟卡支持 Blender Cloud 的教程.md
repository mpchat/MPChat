# 使用 MPChat 虚拟卡支持 Blender Cloud 的教程

## 准备

1. Blender Cloud 概述

   * Blender Cloud 是 Blender 基金会的支持平台

   * 通过订阅支持 Blender 开源项目发展

   * 获得独家教程、资产和云服务

2. 网络环境配置

   * 正常网络连接即可（无需特定国家 IP）

   * 稳定的网络环境

3. MPChat 资金准备

   * 准备€10-15 等值 USDT（月费€9.90，年费€99）

   * 建议多预留资金用于汇率波动

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议€15+）
- 等待网络确认到账
```

* 申请国际 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：选择支持欧元区的国家（如德国、荷兰等）
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（欧元）
- 金额：€11-13（包含可能的手续费）
```

### 第二阶段：Blender Cloud 订阅流程

1. 访问 Blender Cloud 官网

   * 访问 cloud.blender.org

   * 点击"Join Blender Cloud"

2. 选择订阅方案

1) 创建 Blender ID

   * 点击"Create Account"注册新账户

   * 填写邮箱、用户名、密码

   * 完成邮箱验证

### 第三阶段：支付信息填写

1. 支付界面导航

   * 登录后选择订阅方案

   * 点击"Subscribe Now"

   * 进入支付信息页面

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVC: 3位安全码
- Name on card: 建议使用拼音
```

* 账单地址填写

  * Country: 选择 MPCard 发行国

  * 使用对应国家的地址生成器

  * 示例格式（如选择荷兰）：

```plain&#x20;text
Address: Korte Lijnbaanssteeg 1
City: Amsterdam
Postal Code: 1012 SL
Country: Netherlands
```

## 常见问题解答

### 支付相关问题

Q: 支付后没有立即激活怎么办？

A: 通常立即激活，如延迟可联系 support@blender.org

Q: 可以开发票吗？

A: 是的，支付后会收到电子发票

Q: 支持哪些支付方式？

A: 支持信用卡、PayPal 等多种方式

### 内容访问问题

Q: 订阅包含哪些具体内容？

A: 所有 Blender Cloud 上的教程、资产和云服务

Q: 可以下载内容离线使用吗？

A: 大部分教程和资产可以下载离线使用

Q: 订阅有设备限制吗？

A: 可以在多个设备上使用，但同时只能一个设备登录
