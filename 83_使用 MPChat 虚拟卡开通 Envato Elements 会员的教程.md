# 使用 MPChat 虚拟卡开通 Envato Elements 会员的教程

## 订阅前准备

1. 账户准备

   * 准备邮箱注册 Envato 账户

   * 确定创作需求（模板、素材、字体等）

2. 网络环境配置

   * 使用美国住宅 IP（重要成功因素）

   * 确保 IP 干净且地理位置显示为美国

3. MPChat 资金准备

   * 准备 20−50 等&#x503C;*&#x20;USDT*（月费 16.50，年费$198）

   * 建议多预留资金用于验证费用

## 详细操作步骤

### 第一阶段：MPChat 虚拟卡准备

1. USDT 充值

```plain&#x20;text
操作路径：MPChat App → 钱包 → 存款
- 复制USDT充值地址
- 从外部钱包转入足额USDT（建议$50+）
- 等待网络确认到账
```

* 申请美国区 MPCard

```plain&#x20;text
操作路径：MPChat App → MPCard → 申请新卡
- 发行国家：必须选择"美国"
- 卡片类型：虚拟卡（即时生成）
- 安全记录卡号、有效期、CVV码
```

* 资金划转

```plain&#x20;text
操作路径：MPChat App → 钱包 → 划转
- 从：MP钱包（USDT）
- 到：MPCard（美元）
- 金额：$20-25（包含验证费用）
```

### 第二阶段：Envato 账户注册

1. 访问 Envato 官网

   * 使用美国 IP 访问 elements.envato.com

   * 点击"Get Started"或"Start Free Trial"

2. 选择订阅计划

1) 开始免费试用

   * 通常提供 7 天免费试用

   * 试用期无需支付

   * 试用结束后自动扣费

### 第三阶段：支付信息设置

1. 进入支付页面

   * 点击"Start Your Free Trial"

   * 选择订阅周期

   * 进入支付信息页面

2. MPCard 信息填写

```plain&#x20;text
信息源：MPChat App → MPCard → 卡片详情
- Card number: 16位虚拟卡号
- Expiration date: 有效期（MM/YY）
- CVV: 3位安全码
- Name on card: 建议使用拼音
```

1. 账单地址填写（关键步骤）

   * Country: United States

   * 使用专业美国地址生成器

   * 示例格式：

```plain&#x20;text
Address Line 1: 1234 Creative Street
City: San Francisco
State: CA
ZIP Code: 94102
Phone: 生成美国电话号码
```

## 支付验证与会员激活

1. 即时验证

   * Envato 会进行$1 验证扣款（立即退还）

   * 验证卡片有效性和地址匹配度

2. 试用期开始

   * 支付信息验证通过后，7 天免费试用开始

   * 可以无限制下载所有资源

   * 试用期结束前可随时取消

3. 正式订阅

   * 试用期结束后自动开始扣费

   * 按选择的周期（月付/年付）扣款

   * 会员权益正式生效

## 常见问题解答

### 订阅相关问题

Q: 免费试用需要支付吗？

A: 不需要，但需要提供支付信息，试用结束前取消不收费

Q: 可以随时取消吗？

A: 可以随时取消，取消后仍可用到周期结束

Q: 下载的资源有使用限制吗？

A: 有商业使用许可，但不能转售原始资源

Q: 下载次数有限制吗？

Q: 支持哪些文件格式？
